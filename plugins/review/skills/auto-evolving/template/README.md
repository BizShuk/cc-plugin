# 自演化知識引擎 (Auto-Evolving Knowledge Engine)

一個持續發現未知、辯證融合已知、嚴格評判價值、沉澱為堅實基礎的自動化思維框架。

## 使命

萬物之治，始於正名；系統之固，始於知未知。

本引擎以 Cron 驅動的自動循環運作，每次喚醒執行「探・合・衡・沉」四步週期：
1. `探` — 發現工作空間中的一個關鍵盲點
2. `合` — 以四種哲學框架辯證融合
3. `衡` — 四軸評判（系統/業務/認知/反脆弱）
4. `沉` — 沉澱至三層知識基礎或淘汰歸檔

## 哲學根基

推論與合成階段融合四大思維體系：
- 東方哲學脈 — 道家、王陽明、禪宗
- 西方哲學脈 — 蘇格拉底辯證法、Popper 可證偽性、Taleb 反脆弱
- 系統論脈 — 控制論、複雜適應系統、湧現
- 第一性原理 — 回到最基本的事實與約束

## 知識層級

```
公理層 (axioms)     — 多次驗證、核心不動搖
原則層 (principles) — 已驗證、標準記錄
假說層 (hypotheses) — 未驗證、待觀察
淘汰層 (retired)    — 被棄歸檔、避免重蹈覆轍
```

知識攜帶信心度 (0.0-1.0)，經驗證升級、長期未引用則衰減降級。

## 運作模式

- 可作為 Agent System Prompt 單次呼叫
- 可搭配 Cron 自動循環，持續演化
- 重大變更暫停，通知人類批准

## 快速開始

```bash
# 觸發一次執行週期
./scripts/run_cycle.sh

# 執行信心度衰減
./scripts/decay.sh

# 知識升級
./scripts/promote.sh
```
