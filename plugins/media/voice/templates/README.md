# VoxCPM 10 種聲音樣板 (Voice Design)

這 10 種樣板使用 VoxCPM2 的 `Voice Design` 功能：把聲音描述放進文字最前面的括號裡，格式為 `(描述)要唸的文字`，不需要參考音訊。

`比喻`：這就像點咖啡——你不用自己烘豆，只要說「我要一杯溫熱、奶香重、微甜的拿鐵」，機器就調給你。描述越具體，越接近你要的味道。

---

## 參數小常識 (先讀這段)

每個樣板都附兩個參數，當「起點」用，不是鐵則：

- `cfg_value` (引導強度，`classifier-free guidance`)：數值越高 → 越「死守」你的描述，但太高可能出現怪聲；越低 → 越自然但可能偏離描述。常用區間約 `1.5–2.5`。
- `inference_timesteps` (推論步數)：越高 → 細節越精緻但越慢；越低 → 越快但較粗。常用 `10`，追求品質可拉到 `15–20`。

> Voice Design 有隨機性，`同一段建議生成 1~3 次挑最好的`。

## 10 種聲音樣板設定檔

本目錄已將 10 種聲音樣板獨立儲存為 JSON 設定檔，便於動態載入與擴充。以下為預設樣板清單：

1. `01_warm_narrator.json`：溫暖旁白 (Warm Narrator)
   - 用途：有聲書、紀錄片旁白、長文朗讀
   - 描述：`一位四十歲左右的男性,溫暖低沉的嗓音,語速沉穩,帶有故事感`
   - 建議參數：`cfg_value=2.0`, `inference_timesteps=12`
   - 比喻：深夜電台 DJ，讓人想一直聽下去。

2. `02_cheerful_service.json`：活潑客服女聲 (Cheerful Service)
   - 用途：客服、App 語音、教學引導
   - 描述：`一位年輕女性,明亮親切的聲線,語速適中,語氣帶著微笑`
   - 建議參數：`cfg_value=2.0`, `inference_timesteps=10`
   - 比喻：接起電話第一句就讓你心情變好的客服。

3. `03_news_anchor.json`：沉穩新聞主播 (News Anchor)
   - 用途：新聞播報、正式公告、財經摘要
   - 描述：`專業中性的成年聲音,字正腔圓,清晰穩定,語氣客觀`
   - 建議參數：`cfg_value=2.2`, `inference_timesteps=12`
   - 比喻：晚間七點新聞主播，可信、不情緒化。

4. `04_soothing_asmr.json`：療癒助眠 (Soothing / ASMR)
   - 用途：冥想引導、助眠、放鬆內容
   - 描述：`輕柔氣聲的女性,語速非常慢,溫柔放鬆,像在耳邊低語`
   - 建議參數：`cfg_value=1.8`, `inference_timesteps=15`
   - 比喻：深呼吸瑜伽老師，聽著聽著肩膀就鬆了。

5. `05_podcast_host.json`：Podcast 主持人 (Conversational Host)
   - 用途：對談節目、知識型 Podcast、口語內容
   - 描述：`三十多歲、自然口語的主持人聲音,親和有對話感,不死板`
   - 建議參數：`cfg_value=1.8`, `inference_timesteps=10`
   - 比喻：朋友坐你對面聊天，不像在唸稿。

6. `06_children_story.json`：童趣故事 (Children's Story)
   - 用途：兒童故事、繪本朗讀、教育內容
   - 描述：`活潑生動的聲音,情緒豐富,語調誇張可愛,適合說故事給小孩聽`
   - 建議參數：`cfg_value=2.2`, `inference_timesteps=12`
   - 比喻：幼兒園老師說故事，聲音會「演」起來。

7. `07_energetic_ad.json`：廣告促銷 (Energetic Ad)
   - 用途：廣告配音、促銷、開場
   - 描述：`高能量、興奮有說服力的聲音,語速偏快,充滿熱情`
   - 建議參數：`cfg_value=2.3`, `inference_timesteps=10`
   - 比喻：購物台主持人，讓你忍不住想下單。

8. `08_elderly_sage.json`：長者智者 (Elderly Sage)
   - 用途：角色配音、長者旁白、寓言
   - 描述：`一位年長男性,溫厚緩慢的嗓音,帶有歲月與智慧的滄桑感`
   - 建議參數：`cfg_value=2.0`, `inference_timesteps=15`
   - 比喻：電影裡那位什麼都看透了的老爺爺。

9. `09_corporate.json`：企業簡報 (Corporate Presenter)
   - 用途：企業簡報、教學影片、產品介紹
   - 描述：`中性專業的成年聲音,清晰自信,沉穩不急躁,條理分明`
   - 建議參數：`cfg_value=2.0`, `inference_timesteps=12`
   - 比喻：大型發表會上台報告的主管，穩、清楚、有信服力。

10. `10_epic_trailer.json`：電影預告旁白 (Epic Trailer)
    - 用途：電影預告、戲劇旁白、震撼開場
    - 描述：`低沉富磁性的男性嗓音,張力十足,語速緩慢戲劇化,充滿氣勢`
    - 建議參數：`cfg_value=2.3`, `inference_timesteps=15`
    - 比喻：好萊塢預告片那句「In a world…」的聲音。

## 一鍵生成全部 10 種 (批次腳本)

我們提供 `make_voices.py` 批次腳本，它會自動掃描本目錄下的所有 JSON 設定檔並在 `voice_samples/` 資料夾中產出對應的 `wav` 檔。

執行方式：

```bash
python make_voices.py
```

## 怎麼把樣板變成你「專屬」的聲音? (行動計畫)

1. `先全部生成一輪`：用上面的腳本聽 10 種的基準感覺。
2. `挑 1~2 個最接近你要的`：微調描述詞——加上更具體的形容（例如把「年輕女性」改成「二十出頭、聲音偏甜的台灣女性」）。
3. `微調參數`：覺得不夠像描述 → 調高 `cfg_value`；覺得不夠細緻 → 調高 `inference_timesteps`。
4. `固定喜歡的組合`：把描述與參數記下來當你的「品牌聲音」。
5. `要完全一致的特定人聲` → 改用 `Voice Cloning` (丟參考音訊)，或用 `5~10 分鐘` 音訊做 `LoRA` 微調。

---

## 小提醒

- 描述用 `中文或英文都可以`，VoxCPM2 會自動理解。
- 描述越 `具體` 越好：性別、年齡、音色、語速、情緒、口音都可以寫進去。
- 同一句多生幾次挑最好的，是正常用法，不是出錯。
- 商用前請依你的情境做測試與安全評估；嚴禁用於假冒或詐騙。
