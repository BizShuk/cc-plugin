# lsp

`marksman` is a language server for Markdown that provides completion, diagnostics, reference tracking, and folder-wide link management.

- Project Page: <https://github.com/artempyanykh/marksman>
- Installation:

```bash
brew install marksman
```

## Key Features

- Auto-completion (links, headings, and wiki-links)
- Go-to definition and find references for cross-document links
- Diagnostics (e.g. detect broken links and duplicate headings)
- Rename refactoring across multiple files
